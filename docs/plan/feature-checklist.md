## 概述

本文档列出 CryptoQuant 个人量化交易系统的所有功能点，按实现阶段和优先级排序。

### 优先级说明

| 优先级 | 说明 |
|-------|------|
| P0 | **必须实现** - 核心功能，没有就无法上线 |
| P1 | **应该实现** - 重要功能，极大提升用户体验 |
| P2 | **可以实现** - 增强功能，锦上添花 |
| P3 | **将来实现** - 预留功能，待 API 支持 |

### 状态说明

| 状态 | 说明 |
|-----|------|
| ⬜ | 待开发 |
| 🔄 | 开发中 |
| ✅ | 已完成 |
| ⏸️ | 暂停/阻塞 |
